# 什么是中国国际大学生创新大赛 ?

  中国国际大学生创新大赛（原名中国“互联网+”大学生创新创业大赛）是由教育部等12部委和地方省级人民政府共同主办的国家级创新创业赛事。该比赛覆盖人工智能、绿色能源、生物医药等前沿领域，吸引了来自全球一百多个国家和地区的几千所高校、数万个项目参赛，是国际影响力最大的大学生创新创业大赛之一。

  简单来说，就是你有一个可以盈利想法或正在盈利的项目，那么就可以报名参加中国国际大学生创新大赛。
