## 前言

在当今的软件开发环境中，优质的文档是项目成功的关键要素之一。Fumadocs 作为一个现代化的文档解决方案，提供了强大而灵活的功能来帮助开发者构建出色的文档网站。

## 为什么选择 Fumadocs？

### 现代化的开发体验

Fumadocs 基于 Next.js 15 构建，充分利用了 React Server Components 的优势：

- **零配置启动**：开箱即用的配置
- **类型安全**：完整的 TypeScript 支持
- **性能优越**：静态生成和增量构建
- **SEO 友好**：服务端渲染优化

### 丰富的功能特性

1. **MDX 支持**：在 Markdown 中使用 React 组件
2. **主题系统**：浅色/深色模式自动切换
3. **搜索功能**：内置全文搜索
4. **国际化**：多语言支持
5. **响应式设计**：完美适配移动设备

## 快速开始

### 安装

```bash
npm create fumadocs-app@latest
```

### 基础配置

在 `source.config.ts` 中配置文档源：

```typescript
import { defineConfig, defineDocs } from 'fumadocs-mdx/config';

export const docs = defineDocs({
  docs: {
    dir: 'content/docs',
    schema: frontmatterSchema,
  },
});
```

### 内容结构

推荐的目录结构：

```
content/
├── docs/
│   ├── index.mdx
│   ├── getting-started.mdx
│   └── advanced/
└── blog/
    ├── first-post.mdx
    └── updates.mdx
```

## 高级功能

### 自定义组件

Fumadocs 允许在 MDX 中使用自定义 React 组件：

```jsx
import { Callout } from '@/components/ui/callout';

<Callout type="warning">
  这是一个警告提示框
</Callout>
```

### 数学公式支持

通过 KaTeX 支持数学公式渲染：

$$
E = mc^2
$$

### 代码高亮

支持多种编程语言的语法高亮：

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
```
