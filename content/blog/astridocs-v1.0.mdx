## AstriDocs

&emsp;&emsp;AstriDocs（英 /ˈæstri dɒks/，美 /ˈæstri dɑːks/，读作 “AST-ree docks”）是由 Astri + Docs 构成的合成词：Astri 源自 astral/astronomy（词根自希腊语 astron，意为“星体”），寓意灵感、探索与导航；Docs 直指文档与操作手册。合起来表示“一套以星为引的备赛文档体系”，强调为创新创业提供清晰路径、标准流程与可复用模板。统一书写为驼峰体 AstriDocs；一句话定位：Star-guided docs for innovation.
